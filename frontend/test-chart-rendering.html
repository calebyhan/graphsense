Expected Data Flow

  1. File Upload: CSV data is parsed and stored in rawData
  2. Analysis: Backend processes data and returns recommendations
  3. Recommendation Processing: AutoVizAgent processes recommendations and embeds rawData
  4. Chart Configuration: ChartParameterExtractor creates config with data
  5. Rendering: ChartRenderer receives config.data and renders chart
